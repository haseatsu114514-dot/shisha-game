# キャラクタープロフィール詳細

## 主要キャラクター

### 蒸野 はじめ（むしの はじめ）
- **役割**: 主人公
- **スタイル**: 「フラット・リスペクト」
- **説明**: tonariの店員。当初は自分には何もないと思っていたが、誰のシーシャも否定せずフラットにリスペクトする姿勢が、独自の強みとなる。
- **好きなフレーバー**: ミント、アールグレイ
- **成長目安**: 大会を重ねるごとに「適応力」と「調整精度」が向上。

### 炭場 丈一郎（すみば じょういちろう）
- **役割**: tonari オーナー / 師匠
- **年齢**: 42歳
- **スタイル**: 「クラシック・ソウル」
- **説明**: 穏やかで寡黙な店長。その正体は15年前に世界を放浪し、各地に伝説を残した名プレイヤー。
- **好きなフレーバー**: ダブルアップル、パーン
- **特記**: プレイヤーに「煙の優しさ」を説く。

### 翠川 栞（みどりかわ しおり） / みんと
- **役割**: ヒロイン候補 / ライバル（ぺぱーみんと店長）
- **年齢**: 29歳（自称20歳前後だが、はじめにはバレている）
- **スタイル**: 「ガーリー・ビジュアル」
- **説明**: コンカフェシーシャの先駆者。可愛い見た目とは裏腹に、一般投票をハックする計算高さと、裏では丁寧な仕込みを欠かさない努力家。
- **好きなフレーバー**: ストロベリー、ミルク
- **成長目安**: はじめとの交流を通じ、アイドルとしての武器だけでなく、シーシャそのものの深みに目覚めていく。

### 宵野 葉子（よいの ようこ） / アゲハ
- **役割**: ヒロイン候補 / ギャル
- **スタイル**: 「ヴァイブス・ミキシング」
- **説明**: 直感とノリでフレーバーを混ぜる天才肌。理論はないが、吸った人を笑顔にする不思議な魅力がある。
- **好きなフレーバー**: ライチ、グミ系
- **成長目安**: 自分の「なんとなく」が理論化されていく過程で、スランプと成長を繰り返す。

### 鳴切 亮太（なるぎれ りょうた） / なる
- **役割**: 最大のライバル（ケムリクサ店員）
- **スタイル**: 「スイーツ・マニアック」
- **説明**: はじめと同い年の情熱家。お菓子系ミックスに関しては他の追随を許さない。はじめとは全4章を通じて切磋琢磨する。
- **好きなフレーバー**: チョコレート、バニラ
- **成長目安**: 敗北を糧に、より複雑で繊細なフレーバー構成を身につけていく。

### 鈴木 玲二（すずき れいじ） / 零-REI-
- **役割**: ライバル（V系カリスマ）
- **スタイル**: 「カリスマ・ダーク」
- **説明**: 圧倒的な演出力を持つ。しかし、本名は「鈴木玲二」という極めて一般的な名前であり、そのギャップに強いコンプレックスを持っている。
- **好きなフレーバー**: ブラックベリー、ローズ
- **特記**: 演出と味のギャップをどう埋めるかに苦悩している。

### 藤波 創（ふじなみ そう） / ドクター・ケムリ
- **役割**: ライバル（サイエンティスト）
- **スタイル**: 「データ・オプティマイズ」
- **説明**: 全ての工程を数値化・科学的に分析する。
- **成長目安**: AI（SHISHA-9000）の登場により、データを超えた「人間味」の重要性に気づく。

### ナンディ・カルダモン
- **役割**: アジア大会ライバル
- **スタイル**: 「スパイス・ヨガ」
- **説明**: 呼吸法とスパイスミックスの達人。

### スティーブ・デイビス

- **スタイル**: 「グローバル・ジャッジ」

### 王 煙楼（ワン・イェンロウ） / マスター・フーカ

- **スタイル**: 「エターナル・バランス」

### 神崎 竜二（かんざき りゅうじ） / シーシャ組長
