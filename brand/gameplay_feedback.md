# ゲーム性フィードバック：「煙の向こう側」

---

## ■ 現状の評価

### 強み
- **日常パートの設計が堅実**。バイト→ステータスUP→大会で活用、という循環が明快
- **バイトイベントの質が高い**。シーシャ屋あるあるが満載で、プレイヤーが「あるある」と笑えるイベントが多い
- **好感度5段階＋秘伝レシピ報酬**のシステムは、ペルソナ的な「誰に時間を使うか」問題を生む良設計
- **大会ミニゲームの設計が多彩**。15ステップの工程は、シーシャ作りの実工程を忠実に反映している

### 根本的な課題

**1. プレイ時間が短すぎる（現状15〜20分、目標30〜45分）**

現状の7日間＝14行動コマは、1コマあたり1〜2分の操作で終わる。選択→結果テキスト→次の行動、のサイクルが速すぎて「チルな時間」を感じる暇がない。

**2. ノベル/交流要素が薄い**

バイトイベントは「選択→結果」の一発で終わる。キャラとの交流も数行のテキスト→好感度UP。VA-11 HALL-Aのような「会話の中に時間が流れる」感覚がない。

**3. 大会パートの現実フーカーバトルとの乖離**

現状のミニゲーム設計は「工程を一つずつこなす」形式だが、実際のフーカーバトルでは**4人が同時に作り、審査員が吸い比べる**。この「4人同時進行の緊張感」が設計に反映されていない。

**4. 日常パートと大会パートの空気感が同じ**

「チルな日常」と「イケイケな大会」の対比が設計に入っていない。

## ■ プレイ時間を30〜45分にする具体策

### 現状の時間見積もり

| パート | 現状 | 目標 |
|--------|------|------|
| オープニング会話 | 2分 | 4分 |
| 日常パート（7日間） | 10〜12分 | 20〜28分 |
| 大会パート | 3〜5分（未実装） | 8〜12分 |
| エンディング会話 | 1分 | 3〜5分 |
| **合計** | **16〜20分** | **35〜49分** |

### 日常パートの拡充（10分→25分）

**A. バイトイベントを「複数ターンの会話」に拡張**

現状：テキスト1行→選択肢→結果1行（10秒）

改善案：バイトイベントをVA-11 HALL-A風の「複数行会話＋選択肢」に。

```
例：常連客イベント

常連「おう、いつもの」
はじめ「ダブルアップルにミント少し、ですよね」
常連「覚えてくれてたか。嬉しいねえ」
常連「今日はな、いいことがあったんだよ」
はじめ「何があったんですか？」
常連「孫が生まれたんだ」
  → 選択肢：
    A「おめでとうございます！ 少し甘めにしますね」
    B「それは…特別な一杯作りますよ」
常連「……いいシーシャだ。ありがとうな」
```

1イベント30秒→1分30秒に。1シフトで2〜3イベント→1シフト3〜5分。

**B. ライバル交流を「ミニ会話イベント」化**

現状：1行テキスト→好感度UP（5秒）

改善案：Lv1→2、2→3、3→4、4→5それぞれに固有の会話イベント（5〜10行）を用意。

```
なるLv2イベント例：

なる「おーお前か。暇なの？」
はじめ「配達ついでに」
なる「ちょっと聞いてくれよ。新しいミックス作ったんだけどさ」
なる「チョコ4、バニラ3、キャラメル2、ミント3。どう思う？」
  → 選択肢：
    A「ミント多くない？」→「マジ？ 減らしてみるか」（感覚+2）
    B「最高じゃん」→「だろ！？」（なる好感度++）
    C「実際に吸わせてよ」→ 特別演出（洞察+2）
```

1交流イベント1〜2分。1日2回行動で、うち1回を交流に使う→1日4〜5分。

**C. 閉店後の「チルタイム」を追加**

毎日深夜に入る前に、tonariでスミさんと30秒〜1分の短い会話が入る（スキップ可能）。VA-11 HALL-A的な、仕事終わりの一服の時間。

```
Day3夜のチルタイム：

（店内。スミさんが自分用のシーシャを作っている）
はじめ「…今日も一日お疲れ様です」
スミ「……」（煙を吐く）
スミ「大会の準備はどうだ」
はじめ「正直、自信ないです」
スミ「自信なんかいらん。お前の手で作れ」
```

7日×1分＝7分の追加。これだけで日常パートが大幅に肉厚になる。

**D. LIME会話の拡充**

現状：朝のチェック画面だけ。改善案：夜にも「帰宅後にLIMEを確認する」演出を入れる。ライバルのちょっとした独り言（なる：「今日のミックスの出来は7点。もっといける」）がフィードの形で流れる。1日30秒程度だが、キャラクターの厚みが増す。

## ■ 実際のフーカーバトルに沿った大会設計

### 現実のフーカーバトルの流れ

```
1. エントリー受付（テーマ・ルールの発表）
2. 制限時間内に全員が同時にシーシャを作る（15〜20分）
3. 審査員が全員分を吸い比べる
4. 各項目ごとにスコアリング
5. 一般来場者の投票（人気投票）
6. 合計で順位決定
7. 結果発表
```

### ゲームへの落とし込み

```
【大会パートの新フロー】

① テーマ発表
  MCの焚口ショウが大会テーマを発表
  「今日のテーマは"リラックス"！」
  → テーマに合ったフレーバー選択にボーナス
  → EDMのBGMに切り替わる。日常パートとの空気感の断絶

② 準備フェーズ（制限時間制・4人同時進行表示）
  画面を4分割（主人公が手前大、ライバル3人が上部に小さく）
  主人公のパートだけ操作可能。ライバルは自動で進行するが
  画面端で「なるがノートを確認している」「アダムがDAを慎重に詰めている」
  「眠都が観客に手を振った」等のカットインが入る

  1. フレーバー選択（手持ちから2〜3種）
  2. パッキング配合（12gスライダー）
  3. アルミ穴あけ（リズムゲーム・EDM BGMに合わせて）
  4. 炭の準備（タイミングゲーム）
  5. 炭の配置（ドラッグ＆ドロップ）
  6. 蒸らし設定（スライダー）

③ 審査フェーズ（4人の煙を審査員が吸い比べる）
  審査員が1人ずつ、4人の煙を順番に吸う
  各人の煙を吸うときにリアクションが表示される
  「土岐、無言で2口目を吸った」→ 高評価のサイン？
  「ランダム審査員、少し首を傾げた」→ 何かが足りない

  審査フェーズ中にプレイヤーが介入できるポイント：
  - 調整（炭の位置微調整・蓋の開閉）を3回まで
  - 洞察が高いと審査員のリアクションからヒントが読める

④ プレゼン＆トーク
  MC「それでは出場者からコメントをどうぞ！」
  4人が順番に自分のシーシャについて語る
  主人公のターンで選択肢：
    - 技術を語る / 想いを語る / 素直に語る / 攻めて語る
  審査員からの質問に切り返す（制限時間付き）
  観客リアクション→一般投票に影響

⑤ 一般投票
  会場の観客がリアルタイムで投票
  画面にバーグラフが伸びていく演出
  眠都のバーが急速に伸びる（ファン投票力）
  → プレゼンの出来が一般投票に影響

⑥ 結果発表
  MCが緊張感を煽る演出
  パッキーが結果をめくっていく
  専門審査スコア（60%）＋ 一般投票（40%）= 総合点
  4位→3位→2位→1位の順に発表

  ★結果発表時にライバルのリアクション演出が入る
```

### 一般投票の仕組み

```
一般投票スコアの決定要因：

ベース = 魅力ステータス × 2
＋ プレゼンスコア × 1.5
＋ テーマ合致ボーナス（テーマに合った選択で+15）
＋ 秘伝レシピボーナス（使用で+10）
＋ 眠都補正（眠都は常に+30。ファンが多い設定）

比率：専門審査60% ＋ 一般投票40%

→ 主人公は専門審査で勝つタイプ。一般投票では眠都が強い。
　 日常パートで魅力を上げれば一般投票でも勝てるようになる。
```

## ■ その他の改善提案

### 1. 偵察システムの改善（お金不要に）

現状：ライバル店の偵察に💰2,000円かかる。
問題：好感度を上げるだけで情報も手に入る設計なのに、別途お金もかかるのは二重課金。

改善案：偵察は行動コマだけ消費。お金は不要。好感度が上がると追加情報が自動的に入る。

### 2. テーマシステムの導入

大会にテーマがあると、フレーバー選択に戦略性が生まれる。

```
テーマ例（1章）：
「リラックス」→ ミント系・バニラ系にボーナス
「トロピカル」→ マンゴー・パイナップルにボーナス
「クラシック」→ ダブルアップルにボーナス

テーマは大会当日の朝に発表（Day7夜に予告だけ出るとさらに良い）
→ 「明日のテーマは…」でDay7夜の選択肢が変わる
```

### 3. 機材の段階的解放

現状：1章からファンネルボウルとHMSが買える。
改善案：章ごとに新機材が解放される。

```
1章: 標準ボウル（初期）、ファンネルボウル
2章: ターキッシュHMS、高品質炭
3章: シリコンボウル、ロータスHMS
4章: カスタムボウル（特注品）

→ 機材は「大会での最大スコア上限」に影響。
   標準ボウルでも勝てるが、良い機材なら余裕ができる。
```

### 4. リプレイ性

1章クリア後に「別の選択を試す」モチベーションが欲しい。

```
- 好感度MAXにしなかったキャラの秘伝レシピが気になる
- 2周目は別のキャラに時間を使える
- 売却で金策が変わる
- テーマ変動（2周目はテーマがランダム化）
→ 3周くらいで全キャラの秘伝レシピコンプリート
```

### 5. バイトの「忙しい日」にランダム性

現状：忙しい日を選ぶとイベントが3〜5個来る。
改善案：特定の日に「激混み」フラグが立ち、通常の倍のイベントが来る。代わりに収入も倍。

```
Day3（金曜夜）とDay6（土曜）は激混みフラグ
→ イベント5〜7個、収入1.5倍
→ 失敗すると「3卓同時パニック」の高難度イベント発生
→ 成功すると度胸が大幅UP
```

### 6. 「煙の質」ビジュアライズ

プレイヤーが作ったシーシャの煙を画面上にビジュアルで表示する。

```
良いシーシャ → 煙が白く、ゆったりと広がる
普通のシーシャ → 煙が普通
悪いシーシャ → 煙が薄い、または焦げた色
最高のシーシャ → 煙が虹色に輝く（パッキー演出に繋がる）

→ バイト中に作ったシーシャにもこの演出が入ると
   プレイヤーが「上手くなってる実感」を得られる
```

