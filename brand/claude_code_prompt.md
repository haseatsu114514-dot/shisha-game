# Claude Code 向け 統合作業プロンプト

> このファイルの内容を Claude Code のチャットに貼り付けてください。

---

あなたはプロのシナリオライター兼ゲームプランナーです。以下の指示に従い、`shisha-game` プロジェクトのデータ整備を行ってください。

---

## 前提資料（必ず読んでから作業してください）

作業前に以下のファイルを必ず読み込んでください：

1. **`brand/character_profiles.md`** — 全キャラクターのプロフィール（章・役割・スタイル・年齢）
2. **`brand/story_and_structure.md`** — 全4章のストーリー構成とテーマ
3. **`data/characters.json`** — 現在のキャラクターデータ（JSON形式）
4. **`data/dialogue/*.json`** — 各キャラクターの実装済み会話データ（口調の把握に使用）

---

## ゲームのコアコンセプト（重要）

このゲームでは、**主人公がキャラクターと関わっていく中で何かを「吸収」し、ステータスが上がる**というシステムがあります。

「教わる」というより、**「この人と付き合っていて気づいたら自分が変わっていた」という感覚**が大切です。
各キャラクターは深いテーマや哲学を持っており、プレイヤーがその人と深く関わることで「こういう点が勉強になった」と後から気づくような設計にしたいです。

---

## 依頼内容

### 1. 設定・ストーリーの添削

`brand/story_and_structure.md` および `brand/character_profiles.md` を参照し、以下を実施してください：

- **各キャラクターの「生き様」とシーシャ製法のリンク強化**
  どのキャラクターも「なぜ今このやり方でシーシャと向き合っているのか」という動機が、その人の人生と直結するように添削してください
- **第2章ライバル陣（組長・レイ・ドクター・ケムリ・アゲハ）の掘り下げ**
- **矛盾点・穴の指摘**

### 2. `data/characters.json` の更新（上書き出力）

`data/characters.json` を読み込み、以下のフィールドを各キャラクターに追加・充実させて上書き出力してください：

```json
{
  "background_secret": "（添削した裏設定・バックボーン）",
  "growth_stats": {
    "primary": "（主人公が交流で最も伸びるステータス: 技術 / センス / 根性 / 魅力 / 洞察）",
    "secondary": "（サブで伸びるステータス）",
    "notes": "（このキャラと関わってどう変わるか。「教わる」ではなく「気づいたら吸収されていた」という感覚で書くこと）"
  },
  "visual_concepts": [
    "（外見・服装・雰囲気のビジュアルコンセプト案①：イラストレーターに口頭説明できるレベルで具体的に）",
    "（外見・服装・雰囲気のビジュアルコンセプト案②）"
  ]
}
```

**重要な個別設定：**
- **眠都（みんと）** の `background_secret` には「公称22歳だが実際はかなり高齢という設定がある。本人は完全に隠している。」を必ず含める
- **かこ** の好きなフレーバーは `Bombshell（ボムシェル）` と `White Musk（ホワイトムスク）`。VTuberやネットミームが好きな「ホワイトオタク」。シーシャを「吸う」だけでなく「作ること」が好き
- **南雲修二（なぐもしゅうじ）** — 大手シーシャチェーン「シーステーション」の会長兼大会審査員長、45歳

### 3. 技術ドキュメント構成の提案

現在のプロジェクトに `docs/` ディレクトリを設け、開発用 Markdown 資料の目次ツリーを提案してください。

---

## 出力形式

- JSON の上書きは `data/characters.json` ファイルへ直接書き込んでください
- 技術ドキュメントの提案と添削コメントはチャット上に出力してください

---

## トーン・注意事項

- シーシャ特有の「熱管理・煙の質感・フレーバー製法」などを比喩に使い、情緒的かつ専門的なテキストを心がけてください
- ビジュアルコンセプトは「イラストレーターに口頭で説明できる」レベルの具体性（髪型・服装・色・雰囲気・モデル年代のイメージなど）を持たせてください
- `growth_stats.notes` は「○○から学んだ」ではなく「いつの間にか△△が変わっていた」という視点で書くこと
