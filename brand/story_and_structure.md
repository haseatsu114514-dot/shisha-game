シーシャバトル ストーリー大枠

主人公の性格
蒸野はじめ：
  優しい。人の話を聞くのが自然にできる。
  自分に自信がない。「俺には何もない」が口癖に近い。
  でも誰かのために作るシーシャは自然と丁寧になる。
  「何もない」ことが最終的に強みになる物語。

ライバル関係
鳴切（なるぎれ）通称：なる
  はじめと同い年。最初の大会で出会い、世界大会準決勝で激突。
  全4章を通じて共に成長する最大のライバル。
  お菓子系フレーバー特化。

第1章：地方大会編「なんとなく始めた」
テーマ：「俺には何もない」
【起】
  はじめはチルハウスでなんとなくバイトしてる日々。
  スミさんに「大会出ろ。お前の煙は優しい」と言われ、
  よく分からないまま出場を決める。

【承】
  日常パートでライバルと出会う。
  
  なる → 同い年で同じバイトなのに「シーシャが好き」と言い切れる熱量。
          はじめは「俺にはあそこまでの熱がない」と感じる。
  
  アダム → ダブルアップル一本の狂気。
            「自分のシーシャ」を持つ人の凄みを目の当たりにする。
  
  リュウジ → 元暴走族が人生をかけてシーシャに向き合ってる。
              「俺にはそこまでの覚悟がない」とさらに凹む。

  バイトの日々の中で、お客さんの「美味しい」に小さな喜びを感じるはじめ。
  でもそれが「自分の強み」だとはまだ気づいていない。

【転】大会当日
  初めての大会の緊張。自分だけ場違いに感じる。
  アダムの狂気、リュウジの気合い、なるの集中力。
  みんな「自分のシーシャ」を持っている。
  
  はじめは迷いながらも「今の自分にできること」をやる。
  華やかな技術も哲学もないけれど、
  バイトで毎日やってきた丁寧な仕事をそのまま出す。
  
  結果：辛勝。はじめ自身が一番驚いている。

【結】大会後
  なる「……やるじゃん。次は負けねえ」
  → 二人の間にライバル関係が芽生える

  スミさんとの会話：
  スミ「実は俺も昔この大会に出てた。準優勝で終わった」
  スミ「お前に出ろって言ったのは…正直、俺の夢の続きを見たかったからだ」
  スミ「すまん」
  はじめ「……」
  はじめ「いいっすよ。俺も楽しかったし」
  はじめ「でもこれからは、俺は俺の理由で続けます」
  
  → はじめが「誰かに言われたから」ではなく「自分で走る」ことを決める
  → でも「自分のシーシャ」が何かはまだ分からない

第2章：日本大会編「壁にぶつかる」
テーマ：「才能がある人たちの中で、俺はどう戦う？」
【起】
  地方大会優勝で少し自信がついたはじめ。
  だが日本大会の出場者リストを見て愕然。
  レベルが違う。

  きらら → コンカフェシーシャ。一般投票2倍補正。可愛さという武器。
  あげは → 直感型天才。理論で説明できない正解を引く。
  ドクター・ケムリ → 全てをデータで最適化する科学者。
  DJ SMOKE → 音楽とシーシャの融合。リズムゲームが変則化。
  零-REI- → V系カリスマ。きららと一般投票を奪い合う。

【承】
  練習しても追いつかない焦り。
  「俺には"これ"っていう武器がない」

  バイトの日々で失敗もする。
  3卓同時の炭替えでパニック。お客さんに「前より味落ちたね」と言われる。
  
  でもスミさんが「失敗は経験値だ。バイトでの1回のミスは大会での1回の正解になる」。

  あげはとの交流で転機が訪れる。
  天才だと思っていたあげはが実は不安を抱えていた。
  あげは「理論がないから、いつか当たらなくなるんじゃないかって怖い」
  はじめ「でもあげはさんのシーシャ、吸った人みんな笑ってましたよ」
  あげは「……あんた、なんかずるいね。そうやって人の核心突くの」

【転】大会
  きららの一般投票力、ケムリの完璧なデータに追い詰められる。
  だがはじめは「捜査パートで集めた情報＋バイトで培った対応力」で勝ち上がる。
  派手な武器はないが「相手と審査員をよく見てる」ことが武器になり始める。

【結】
  あげはの本質を知る。天才じゃなく「自分のヴァイブスを信じる強さ」の人。
  はじめ「自分を信じる」ことを少しだけ覚える。
  でもまだ「自分のシーシャ」の答えは出ない。

  なるも勝ち上がり。「全国の舞台で会おう」と約束。

第3章：アジア大会編「世界の広さを知る」
テーマ：「シーシャは文化だ。俺は何も知らなかった」
【起】
  アジア大会はイスタンブール開催。はじめにとって初めての海外。
  トプハーネ地区のナルギレカフェ。500年の歴史。
  「シーシャってこんな深い世界だったのか」と圧倒される。

【承】
  世界レベルのライバルたち。全員が「シーシャ観」を持っている。
  
  フーカ老師 → 「陰陽五行の理。お前の煙には"命"がある」
  ヴォルク → 努力の極限。全工程完璧。一切のミスなし。
  ナンディ → 身体も楽器。呼吸法で煙をコントロール。
  エミル → 500年の伝統。「新しいだけでは深みは出ない」
  ノマド → 世界を旅してきた男。「お前は空っぽだ。でもそれが武器だ」

  ノマドの言葉がはじめの心に刺さる。
  「空っぽだからこそ何でも入る。お前は全部吸収できる」

  エミルの伝統式を体験し、シーシャの歴史の重みを知る。
  「俺、シーシャのことを何も分かってなかった」

【転】大会
  なるがヴォルクに負ける。完璧な相手に打ち勝てなかった。
  なるが初めて泣く。

  なる「悔しい。悔しいよ、はじめ」
  なる「……世界大会では絶対勝ち上がる。お前と準決でやる」

  → ゲーム前半最大の感動ポイント
  → ライバルの涙がはじめの覚悟を固める

  はじめはアジア大会を勝ち上がる。
  各国のシーシャ文化を「空っぽ」の自分に吸収しながら。

【結】
  大会後、エミルがはじめに近づく。
  エミル「お前のシーシャは……あの人に似ている」
  はじめ「あの人？」
  エミル「40年前、この街で一度だけ出会った日本人の女性。
          彼女のナルギレは…今でも忘れられない」
  
  → 香月トキ（隠しボス）の伏線。
  → はじめは意味が分からないまま帰国。
  → プレイヤーには伏線が見える。

第4章：世界大会編「自分のシーシャ」
テーマ：「全部背負って、最後に出す一杯」
【起】
  世界大会はドバイ開催。
  シェイク・アル＝ナルギーレの純金機材。住む世界が違う。
  
  でもドバイのスーク（市場）で安い無名のフレーバーを見つけ、
  「高い機材じゃなくても、俺の手で最高の一杯を作る」と決意。

【承】
  世界大会のライバルたちとの交流。

  シェイク → 変装して路地裏のカフェで一人シーシャを吸ってるのを目撃。
              「ここのシーシャが一番落ち着く。王宮より」
              → 王族にも人間味がある

  チャド → 「味はいいけどショーマンシップが足りねえ」
            → プレゼン力を教えてくれる意外な師匠

  ダ・シルヴァ太陽 → 「シーシャはフェスタだ！楽しくなけりゃ意味がねぇ！」
              → 純粋な楽しさを思い出させてくれる

  SHISHA-9000 → AIの完璧さ vs 人間の不完全さ
                 → 「心のあるシーシャ」とは何かを考えさせられる

【転-1】SHISHA-9000戦
  AIのほぼ完璧なスコア。だがプレゼンで壊滅。
  はじめが勝った後、開発者が「やはり心が必要なのか」と問う。
  
  はじめ「心っていうか…"誰かに吸ってほしい"って
          思いながら作ってるかどうか、じゃないですか」
  
  → 「自分のシーシャ」の答えに近づく瞬間

【転-2】準決勝：はじめ vs なる ★ゲーム最大の感動パート
  最初の地方大会で出会い、共に成長してきた二人の決着。
  
  なるはアジアでの敗北から死ぬほど努力して世界大会に戻ってきた。
  はじめの全試合データを分析。弱点を完璧に突いてくる。
  
  なる「お前のこと、誰より見てきたからな。お前の癖、全部知ってる」
  
  でもはじめが「なるの分析にないこと」をやる。
  旅で出会った全ライバルの技を自分なりに融合した即興の一杯。
  
  僅差で勝利。
  
  なる「………」
  なる「…最高のシーシャだったよ。決勝、絶対勝てよ」

【転-3】決勝：はじめ vs シェイク
  純金装備 vs 普通の装備。
  シェイクが完璧なシーシャを作る。審査員全員が高得点。
  
  はじめが出したのは「チルハウスでいつも作ってるシーシャ」のアレンジ。
  特別な機材じゃない。特別なフレーバーでもない。
  でも今までの全てが込められている。
  スミさんの教え、なるとの約束、世界中のライバルの哲学。
  そして何より「目の前の人に吸ってほしい」という気持ち。
  
  土岐「…………悪くない」（最高評価。会場がどよめく）
  パッキーが一瞬沈黙。煙が虹色になる。
  
  はじめ、優勝。

【結】世界大会後
  シェイクに屋敷に招かれる。最高のシーシャを振る舞われる。
  
  シェイク「実はな…私がシーシャを始めたきっかけは日本人だ。
            20年前、ドバイの路地裏のカフェで出会った名もない店員。
            あの男のシーシャだけは忘れられない」
  
  シェイク「彼の名は……炭場丈一郎」
  
  はじめ「…………スミさん」
  
  → スミさんは日本大会準優勝の後、傷心で世界を放浪。
    ドバイの小さなカフェで働いていた時期がある。
    シェイクにとってスミさんは「シーシャの原体験」だった。
  
  → 師弟の系譜が世界規模で繋がる

  帰国後、チルハウスでスミさんと向き合う。
  
  はじめ「スミさん。シェイクさんから聞きました」
  スミ「……そうか」
  はじめ「スミさんが俺に"お前の煙は優しい"って言ってくれた意味、
          やっと分かった気がします」
  スミ「…分かったか」
  はじめ「俺のシーシャは、目の前の人のために作るシーシャです」
  スミ「……ああ。それでいい」

テーマの着地
「自分のシーシャとは何か？」に対するはじめの答え：

  「目の前の人が、今一番必要としてる一杯を作ること」

  派手な技術でもない。哲学でもない。データでもない。
  ただ、相手の話を聞いて、その人のための煙を作る。
  それがバイトで培った「優しさ」の究極形。

  シーシャ屋で働いてる人が
  「俺たちがやってることって、これだよな」
  って思える結論。

打ち上げ（修羅場）
世界大会優勝後、チルハウスで打ち上げ。
好感度MAXのヒロイン候補が複数いると修羅場発生。

対象：つむぎ、きらら、あげは
好感度MAX1人→自然にその人とのイベント
好感度MAX2人以上→修羅場

全員ルート（逃げの選択肢）もあり。
はじめが「とりあえず全員でシーシャ吸いませんか」と言って
なるが遠くから「あいつ終わったな笑」と実況する。

隠しボス：香月トキ
全クリ後、チルハウスに割烹着のおばあちゃんが客として来る。

スミさんが震える。
スミ「…お客様…まさか…」
トキ「あらあら丈一郎くん、大きくなったわねぇ」

→ スミさんが子供の頃、トキのシーシャを吸ったのが原体験だった。
→ トキ → スミさん → はじめ。3世代のシーシャの系譜が完成。
→ エミルが語った「40年前の日本人女性」もトキだった。

トキとの対決：全ステ「？？？」。弱点なし。
パッキーだけがトキに敬語を使う（唯一）。

勝っても負けても：
トキ「あんた、いい煙出すわ。これからもいい煙、出してあげてね」

エンディング：
はじめがチルハウスでいつも通りバイトしてる。
客が来る。

はじめ「いらっしゃいませ。今日はどんな気分ですか？」

→ 最初と同じ日常。でもはじめの中身は全く違う。
