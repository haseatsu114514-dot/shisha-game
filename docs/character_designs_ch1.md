# 第1章 キャラクターデザイン案

本ドキュメントは、ゲーム第1章に登場する主要キャラクター（はじめ、スミ、アダム、きらら、なる）のキャラクターデザイン案をまとめたものです。
各キャラクターの性格、設定、シナリオ上の役割を考慮し、視覚的な特徴を提案します。

## 1. 蒸野 はじめ (Hajime Mushino) - 主人公
*   **年齢:** 21歳
*   **役割:** 主人公、成長する新人シーシャプレイヤー
*   **キャッチコピー:** 「真っ直ぐな瞳の観察者」
*   **外見的特徴:**
    *   **髪:** 黒髪、少し短めでさっぱりしているが、前髪は少し長めで目にかかりそう（観察眼の象徴）。
    *   **目:** 意思の強さを感じさせる瞳。色はブラウン系。
    *   **体型:** 標準的な大学生の体型。細身だが健康的。
*   **服装:**
    *   **普段着:** パーカーやシャツにデニムなど、大学生らしいシンプルな服装。色は白やグレー、淡い青など、清潔感のある色。
    *   **仕事着（チルハウス）:** 店のエプロン（ダークブラウンか黒）。下には白シャツ。
*   **備考:**
    *   プレイヤーが感情移入しやすいよう、あまり個性が強すぎないデザイン。
    *   「観察する」というアクションが重要なため、目元の表情が分かりやすいようにする。

## 2. 炭場 丈一郎 (Joichiro Sumiba / Sumi) - 師匠
*   **年齢:** 38歳
*   **役割:** メンター、チルハウス店主
*   **キャッチコピー:** 「静寂を纏う熟練者」
*   **外見的特徴:**
    *   **髪:** 黒髪に少し白髪が混じっている。オールバックか、軽く後ろに流している。
    *   **顔:** 整えられた口髭とあご髭。目じりに笑い皺があるが、真剣な時は鋭い眼光。眼鏡をかけている（作業中は外すことも）。
    *   **体型:** 背筋が伸びている。細身だが筋肉質（過去に何かやっていた雰囲気）。
*   **服装:**
    *   **仕事着:** 襟付きのシャツ（白や紺）にベスト、タイ、サロンエプロン。バーテンダーのようなきちんとした装い。
    *   **アクセサリー:** 腕時計（アナログ、革ベルト）。
*   **カラーイメージ:**
    *   **メイン:** ダークブラウン（コーヒー、木材、落ち着いた店内の色）
    *   **サブ:** 深い緑（落ち着き、癒し）
*   **備考:**
    *   「大人の余裕」と「隠された過去」を感じさせるデザイン。
    *   常に落ち着いていて、所作が美しい。

## 3. 林檎堂 アダム (Adam Ringodo) - ライバル (Chapter 1)
*   **年齢:** 27歳
*   **役割:** 職人気質のライバル、「Apple Core」店主
*   **キャッチコピー:** 「寡黙な求道者」
*   **外見的特徴:**
    *   **体型:** 185cm以上の長身。筋肉質で大柄。威圧感がある。
    *   **顔:** 強面。眉間に皺が寄りがち。無精髭。
    *   **髪:** 短髪、あるいはバズカット。色は黒か濃い茶色。
*   **服装:**
    *   **仕事着:** 使い古された厚手のキャンバス地のエプロン（汚れが味になっている）。Tシャツ（無地、黒やグレー）。
    *   **手元:** 腕まくりをしており、腕には火傷の跡や筋肉の筋が見える。
*   **アイテム:**
    *   画面の割れたスマートフォン。
*   **カラーイメージ:**
    *   **メイン:** 赤（リンゴ、情熱）
    *   **サブ:** 黒（無骨さ、炭）
*   **備考:**
    *   一見怖いが、実は優しいというギャップを表現。
    *   「ダブルアップル」へのこだわりから、赤をアクセントに取り入れる。

## 4. 水無月 きらら (Kirara Minazuki) - ライバル (Chapter 1)
*   **年齢:** 24歳
*   **役割:** アイドル的なライバル、SNSインフルエンサー
*   **キャッチコピー:** 「華やかなる努力家」
*   **外見的特徴:**
    *   **髪:** 明るい色のロングヘア（ピンクベージュやミルクティー色）。巻き髪。
    *   **顔:** ばっちりメイク。大きな瞳、長いまつ毛。常に笑顔。
    *   **体型:** スタイルが良い。
*   **服装:**
    *   **仕事着:** フリルのついた可愛いエプロン（ピンクやパステルカラー）。インナーはオフショルダーなど、少し露出のある服で華やかさを演出。
    *   **ネイル:** 派手なネイルアート（ただし、シーシャ作りには邪魔にならない長さ、あるいはボロボロになっている指先が見える演出）。
*   **カラーイメージ:**
    *   **メイン:** ピンク（可愛さ、華）
    *   **サブ:** ゴールド（輝き、成功への執着）
*   **備考:**
    *   「可愛い」で武装しているが、ふとした瞬間に見せる真剣な表情（ギャップ）が重要。
    *   手元だけは職人のように荒れていたり、タコができていたりすると良い。

## 5. 鳴切 亮太 (Ryota Narukiri / Naru) - ライバル (Chapter 1)
*   **年齢:** 21歳
*   **役割:** 同い年のライバル、分析家
*   **キャッチコピー:** 「理詰めの感覚派」
*   **外見的特徴:**
    *   **髪:** 明るめの茶髪かアッシュ。少し跳ねている（寝癖っぽい、あるいは無造作ヘア）。
    *   **顔:** 童顔。目が大きく、好奇心で輝いている。
    *   **表情:** よく笑い、よく喋る。表情豊か。
*   **服装:**
    *   **普段着/仕事着:** ストリート系のカジュアルファッション。オーバーサイズのパーカーやTシャツ。キャップを被っていることも。
    *   **アクセサリー:** 大きめのヘッドフォンを首にかけている（音楽好き、あるいは集中するため）。眼鏡をかけることもある（分析モード時）。
*   **カラーイメージ:**
    *   **メイン:** 青（知性、クールなフレーバー、ブルーベリー）
    *   **サブ:** 黄色（活発さ、電気、ひらめき）
*   **備考:**
    *   主人公と同年代で、最も親しみやすいライバル。
    *   「匂い」に敏感なため、鼻を触る仕草などが似合う。
