# キャラクター設定案：みんと（みんちゃん）

- **源氏名**: みんと（みんと） / 愛称：みんちゃん
- **年齢**: 20代前半
- **所属**: コンカフェ系シーシャ「ぺぱーみんと」の看板店員
- **得意フレーバー**: フルーツ系、フローラル系の華やか・映え重視ミックス

## キャラクターコンセプト（改訂版）
**「コンカフェ嬢として絶大な人気を誇る、距離感の近い可愛い店員」**

ただの地元のアイドルというよりは、**「コンカフェ嬢（コンセプトカフェのキャスト）」としてのプロ意識と人気**を持つキャラクターへと設定をシフトする。
- **特徴的な振る舞い**:
  - お客さんに対する「疑似恋愛的な距離感の近さ」や「可愛く見せる振る舞い」はプロのコンカフェ嬢としての性質。
  - シーシャ屋としての「ぺぱーみんと」も、コンカフェ的なノリ（チェキや可愛いドリンクなど）を重視した空間。

### 裏設定：実家の喫茶店と精肉店跡地からの逆転
- **過去の経緯**: 実家はもともと喫茶店を経営（看板メニューはチョコミントデザート）していたが、親の都合で自己破産。店を手放すことになり、その跡地にはただの「精肉店」が建ってしまった。
- **逆境からの生還**: 悲しみを胸に秘めつつ、チョコミントの思い出から自らを「ミント」と名乗り、別のシーシャ屋で不本意ながら働き始める。しかし大人になってから死ぬ気で自力でお金を稼ぎ、なんとその「精肉店の物件」を見事買い戻した。
- **コンカフェ化の理由**: 買い戻した店をシーシャ屋として経営するにあたり、「どうすれば客が一番お金を落とすか」を徹底的に計算した結果、自身の童顔を活かした「コンカフェシーシャ」の営業スタイルに行き着いた。
- **26歳とビジネスライク**: 経営を取り戻すのに時間がかかったため、実年齢は「26歳」とややいっているが、コンカフェ嬢としてキャピキャピした態度で若作りをしている。本気で商売をしているため強かでビジネスライクな思考を持っており、「可愛いだけの見掛け倒し」と舐められることを何より嫌う。
  - シーシャの味自体にも手を抜いておらず、「可愛い空間＋甘くて美味しい煙」で癒やしを提供するエンターテイナー。

## 好感度によるシナリオ推移（1章内でMAX到達可能）

### 序盤（好感度1〜2）：コンカフェ嬢としての「営業」
「ぺぱーみんと」を訪れた主人公を、持ち前のコンカフェ嬢としての接客術（至近距離での上目遣いやあざとい振る舞い）で歓迎する。
- 初対面から「はじめくん！」とフレンドリーに呼び、可愛いフレーバーを勧めてくる。
- *「いらっしゃいませーっ！ 新人さんだ！ わーい、はじめくんが遊びに来てくれて、みんちゃんすっごく嬉しいなっ♡」*

- 「コンカフェのメイドが出すおもちゃシーシャ」と舐められることもあるが、味へのこだわりは本物。
- *「『コンカフェの煙なんて』って言う人もいるけど……だからこそ、味でも絶対にガッカリさせたくないんだー！ ほら、この甘さ、完璧でしょ？」*

### 終盤（好感度5）：心からのアプローチ（恋人ルートへの布石）

- 他の客には見せないような、少し照れたような素の表情や本音を見せる。
- *「ねえはじめくん。……あのさ、営業じゃなくてね？ 今度お店とか関係なく、普通に遊びに行かない？ ……なんて、みんちゃんの本気、伝わってるかな？」*
