# キャラクター設定案：アダム（旧・留学生設定からの改訂版）

## 基本プロフィール
- **本名**: 鈴木（または普通の日本名。本人は頑なに「アダム」と名乗る）。
- **通称**: アダム
- **年齢**: 20代前半〜半ば
- **所属**: シーシャ専門店「Eden」の店長（またはオーナー）
- **得意フレーバー**: ダブルアップル（これしか吸わないし、これしか作らない）

## キャラクターコンセプト
**「ダブルアップルを愛しすぎたあまり、自分を“最初の人類（アダム）”だと思い込んでいるヤバい日本男児」**

以前の「真面目な外国人留学生」という設定を撤廃し、**「シーシャ狂い・ダブルアップル狂いの変な日本人」**というエッジの効いたキャラクターに変更する。
- **特徴的な振る舞い**:
  - 外見や顔立ちは完全に日本人だが、なぜか常にカタコト、あるいは芝居がかった口調で話す。
  - すべての例え話が「リンゴ」や「エデンの園」にまつわるものになる。
  - 傍から見るとただの変人だが、**実はとてつもなく真面目な求道者**である。

## アダムの底にある「真の理由（裏設定）」
彼が狂信的なまでにダブルアップルしか作らないのには、真面目すぎる理由がある。
**「ダブルアップルは、シーシャの歴史上もっとも古く、最初に発明された伝統的なフレーバーである。その『原点』を完全に極め切るまでは、決して他のフレーバーに手を出すべきではない」**という独特でストイックな価値観を持っている。
彼の中でダブルアップルの探求は未だ道半ばであり、「ずっと極めるためには時間が要る」ため、他の味に行くことができない不器用な職人なのである。

## 好感度によるシナリオ推移（1章内でMAX到達可能）

### 序盤（好感度1〜2）：狂信者との遭遇
「Eden」を訪れた主人公を歓迎するが、その偏執的なダブルアップル愛と奇妙なテンションに主人公はドン引きする。
- 何を言っても「リンゴ」の話題しか通じない変人として描く。
- 他の店舗（tonariなど）のミックススタイルを「純粋なるリンゴへの冒涜」と呼ぶ。
- *「オオ、迷える子羊ヨ！ トゥナイトも『禁断の果実（ダブルアップル）』の甘き誘惑に囚われに来タノカ！」*

### 中盤（好感度3〜4）：狂気の裏にある「求道者」としての境地
好感度が3を超えたあたりから、彼がただの狂人ではなく、**シーシャの歴史を愛し、原点をリスペクトする超・真面目な男**であることが垣間見えてくる。
- *「何故、リンゴだけを吸うのカト……？ フフ、愚問ダネ。ダブルアップルは、人類が最初に手にした偉大なる伝統……その『原点』を極めずして、どうして他の果実を語れヨウカ……！」*
- その狂気にも似たストイックさが、常軌を逸して美味い至高のダブルアップルを生み出していることに主人公も気づき、畏敬の念を抱く。

### 終盤（好感度5）：アダムからの「リンゴの洗礼」と親愛
主人公のブレない芯の強さやシーシャへの姿勢を認め、「同胞」として扱うようになる。
- ダブルアップルの美味しい作り方のコツを伝授してくれたり、彼なりの独特な言い回しで最大限の賛辞を送ってくる。
- *「ユーは……リンゴに愛されし存在。マイ・ブラザーだ。さあ、共に真理の園で永遠のリンゴを追究しようゾ……！」*
